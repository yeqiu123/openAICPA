# Joy Match Quest

一个面向 Android 的三消闯关游戏工程，玩法参考经典开心消消乐方向：关卡推进、目标收集、丰富道具、连锁消除和精致糖果棋盘。

## 当前进度

- Android 原生工程骨架，单 Activity 竖屏游戏入口。
- 8x8 三消棋盘、交换、消除、下落补充、连锁消除。
- 120 个关卡，包含分数、收集、冰块、蜂蜜、石块等多类目标。
- 道具包含锤子、炸弹、重排、十字、同色消除。
- 四连/五连特殊棋子、连击反馈、选关地图、星级和本地进度保存。

## 开发

使用 Android Studio 打开项目根目录，等待 Gradle 同步后运行 `app` 模块。

如果不想打开 IDE，也可以执行 `gradlew.bat assembleDebug`。
